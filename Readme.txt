โปรเจคนี้เป็นส่วนหนึ่งของการศึกษา
โปรเจกต์นี้เป็น ระบบสหกรณ์ออมทรัพย์จำลอง (Cooperative Banking System) ด้วย PHP SQL HTML CSS JS ที่มีฟีเจอร์หลักดังนี้
ระบบล็อกอิน	รองรับทั้งผู้ใช้งานและผู้ดูแลระบบ
ATM จำลอง	ฝาก-ถอน เช็คยอด ตรวจสอบจำนวนธนบัตร
บัญชีเงินฝาก ฝาก ถอน โอนเงิน แสดงรายการย้อนหลัง
การจ่ายปันผล แสดงข้อมูลการปันผลให้สมาชิก
จัดการผู้ใช้ สมัครสมาชิก แก้ไขข้อมูล ลบบัญชี
แผนที่	แสดงตำแหน่งสาขาหรือ ATM (คาดว่าจะใช้ map.php)
ระบบอัปโหลดรูป	สำหรับอัปโหลดโปรไฟล์หรือเอกสารแนบ
ระบบรายการเดินบัญชี	ดูรายการธุรกรรมย้อนหลัง
